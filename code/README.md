# Documentación

Pasos para ejecucion:
1) Generar las instancias deseadas de sorting o multiplicacion de matrices ejecutando desde el nivel base del repositorio

2) Para ejecutar los algoritmos hay que acceder por consola hasta el nivel de sorting o matrix_multiplicaction es decir: "./code/sorting/" por ejemplo y colocar en la consola "make run", esto compilara y ejecutara los archivos calculando las metricas correspondientes del siguiente paso

3) Deberian generarse automaticamente los archivos de output y de metricas en las secciones "data/array-output" y "data measurements"  amodo de ejemplo.

4) Finalmente ejecutar plot_generator.py para obtener los graficos correspondientes al tipo de algoritmo que buscamos analizar. Esta parte hay que realizarla al mismo nivel que el paso 2)

5) make clean limpia los archivos generados para evitar un uso excesivo en disco.

## Entrega

La entrega se realiza vía **aula.usm.cl** en formato `.zip`.
